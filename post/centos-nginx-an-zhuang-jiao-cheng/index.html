CentOS Nginx 安装教程

 2021 年 01 月 22日

1. 安装依赖

yum -y install gcc pcre-devel zlib zlib-devel

2. 选择安装

yum -y install openssl openssl-devel

3. 下载nginx

Nginx 下载官网

使用命令： wget http://nginx.org/download/nginx-1.16.1.tar.gz

4. 解压

tar -zxvf nginx-1.16.1.tar.gz

5. 运行安装

进入解压后的目录，运行 ./configure --prefix=/usr/local/nginx

接下来执行 make

继续执行 make install

然后安装成功！

6. nginx 常用命令

/usr/local/nginx/sbin/nginx -t # 测试配置文件

/usr/local/nginx/sbin/nginx # 启动服务

/usr/local/nginx/sbin/nginx - s stop # 停止服务

/usr/local/nginx/sbin/nginx - s reload # 重启服务

ps -ef | grep nginx # 查看进程

kill -HUP 进程号 # 平滑重启

7. nginx 自启文件配置

vim /usr/lib/systemd/system/nginx.service

[Unit]
Description=nginx
After=network.target
  
[Service]
Type=forking
ExecStartPre=/usr/local/nginx/sbin/nginx -t -c /usr/local/nginx/conf/nginx.conf
ExecStart=/usr/local/nginx/sbin/nginx -c /usr/local/nginx/conf/nginx.conf
ExecReload=/bin/kill -s HUP $MAINPID
ExecStop=/bin/kill -s QUIT $MAINPID
PrivateTmp=true

[Install]
WantedBy=multi-user.target

8. 配置域名映射

sudo vim /usr/local/nginx/conf/nginx.conf

在最后一个结束花括号前增加：include vhost/*.conf; # 后面有分号

在conf目录下创建vhost文件夹，以后配置文件就可以编写在vhost中了。

conf 示例：

# 配置映射到某一文件夹
server {
listen 80;
autoindex off;
server_name img.xxx.com; # 映射ip
access_log /usr/local/nginx/logs/access.log combined;
index index.html index.htm index.jsp index.php;
#error_page 404 /404.html;
if ( $query_string ~* ".*[\;'\<\>].*" ){
   return 404;
}

location / {
   root /ftpfile/; # 映射的目标文件夹
   add_header Access-Control-Allow-Origin *;
}
}

# 配置映射到某一网址
server {
 default_type 'text/html';
 charset utf-8;
 listen 80;
 autoindex on;
 server_name www.xxx.com;  # 映射ip
 access_log /usr/local/nginx/logs/access.log combined;
 index index.html index.htm index.jsp index.php;
 #error_page 404 /404.html;
 if ( $query_string ~* ".*[\;'\<\>].*" ){
     return 404;
 }

 location / {
     proxy_pass http://127.0.0.1:8080;
     add_header Access-Control-Allow-Origin *;
 }
}
